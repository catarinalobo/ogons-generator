Some Subclasses of Orthogonal Polygons

The subclasses of orthogonal polygons that our library currently provides are described below. The polygons we show have been generated by our program.


Row-Convex Orthogonal Polygons

An orthogonal polygon is row-convex if it is monotone with respect to a vertical line.

An orthogonal polygon is monotone w.r.t. a line L if its intersection with any line L' perpendicular to L has at most one connected component
(i.e., it is a segment, a point or an empty set).

Column-Convex Orthogonal Polygons

An orthogonal polygon is column-convex if it is monotone with respect to an horizontal line.

Convex Orthogonal Polygons

A convex orthogonal polygon is an orthogonal polygon that is both row-convex and column-convex.

Path Orthogonal Polygons

A path orthogonal polygon is an orthogonal polygon for which the dual graph of its rectilinear partition is a path graph.

The rectilinear partition is obtained by adding all the rectilinear (horizontal and vertical) cuts (or chords) incident to the reflex vertices of the polygon.
This means that we extend all edges towards the interior of the polygon until they reach the boundary again. The dual graph of the partition is a graph
whose vertices correspond to its pieces (faces of the decomposition) and whose edges connect pieces that share an edge.

Spiral Orthogonal Polygons

A spiral orthogonal polygon is an orthogonal polygon whose boundary can be divided into two chains, a reflex and a convex vertex chain.
